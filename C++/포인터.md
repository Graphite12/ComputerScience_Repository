## 포인터란?

포인터는 메모리 주소를 저장하는 변수를 말한다.

`&`: 참조, reference
`*`: 포인터

### 포인터 선언
```cpp
// 자료형 (포인터) 변수 = 값(optional); 
int *a, *b, c;          // a, b는 포인터, c는  int
char *text = 'string';  // text는 포인터
```

#### 주소 연산자를 통한 포인터 초기화
```cpp
int num = 123;

/* 방법1 */
int *pNum;
pNum = &num;

/* 방법2 */
// e
int *pNum2 = &num;

```
